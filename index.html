Getting Started

To run the chess application on your computer:

  1. Download both files using the buttons above
  2. Make sure you have Python 3.6 or higher installed
  3. Run the installation script: python install_and_run.py
  4. The script will install all required dependencies and launch the application

Alternatively, you can run the main application directly: python main.py

Play against AI

Challenge the computer opponent with different difficulty levels or play against another human player.

Customizable Pieces

Upload your own chess piece images or use the provided standard pieces.

Multiple Board Styles

Choose between classic, blue, and green board styles for a personalized gaming experience.

Custom Chess Pieces

This application can use custom chess piece images from your desktop at: C:/Users/user/Desktop/chess/images/

The expected file names are: wK.png (White King), bQ.png (Black Queen), etc.
